# Terminology Registry

यह registry समान दिखने वाले शब्दों को बिना प्रमाण समानार्थी मानने से रोकती है। यह dictionary नहीं है; प्रत्येक chapter अपने source-specific usage की citation देगा।

| शब्द | IAST | कार्यशील label | उपयोग-नियंत्रण |
| --- | --- | --- | --- |
| मण्डल | maṇḍala | context-dependent circle, group, division, or diagram | किसी एक meaning को सभी corpora पर लागू न करें। |
| ब्रह्माण्ड | brahmāṇḍa | Puranic cosmological term | modern “universe” का स्वतः equivalent नहीं। |
| लोक | loka | world/realm; context-dependent | astronomical location मानने के लिए अलग evidence चाहिए। |
| भू-मण्डल | bhū-maṇḍala | Puranic cosmographic expression | modern Earth map नहीं। |
| द्वीप | dvīpa | island/continental division in context | modern continent से स्वतः न मिलाएँ। |
| मेरु | Meru | cosmographic/sacred mountain term | terrestrial mountain या axial coordinate तभी जब source support करे। |
| ज्योतिर्मण्डल | jyotir-maṇḍala | luminous/celestial domain, context-dependent | modern stellar system का fixed translation नहीं। |
| कल्प | kalpa | large time unit in specific traditions | scientific geological/cosmological duration से अलग define करें। |
| मन्वन्तर | manvantara | traditional temporal division | unit, text और calculation स्पष्ट करें। |
| युग | yuga | age/period in source-specific system | colloquial “era” से अलग technical use हो सकता है। |
| आकाश | ākāśa | space/ether/sky, context-dependent | physical vacuum का automatic synonym नहीं। |
| अन्तरिक्ष | antarikṣa | intervening region/atmospheric-celestial domain | modern outer space के साथ only contextually compare करें। |
| ब्रह्मन् | brahman | philosophical absolute, context-dependent | ब्रह्मा (Brahmā) से अलग रखें। |
