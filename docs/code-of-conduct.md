# Code of Conduct

हम शोध, असहमति और धार्मिक/वैज्ञानिक दृष्टियों पर सम्मानजनक संवाद का स्वागत करते हैं। उत्पीड़न, व्यक्तिगत आक्षेप, धार्मिक घृणा, और बिना स्रोत के भ्रामक दावे स्वीकार्य नहीं हैं।

घटना की सूचना repository maintainers को निजी रूप से दें। योगदानकर्ता से सुधार, अस्थायी प्रतिबंध या स्थायी प्रतिबंध सहित उचित कार्रवाई की जा सकती है।
